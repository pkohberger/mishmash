Overview of APP

A-List is a comprehensive fully-secure app-based list management tool for the event industry, giving users complete control of any type of patron lists within their business; not only in the creation process, but in real-time as an event is happening.


Designed by event industry professionals with over 25 years of experience running multiple types of event, our breakthrough software allows a user to create an unlimited number of events, an unlimited amount of lists and sub-lists within an event, and assign read-only or admin privileges as needed to as many users as they like. From the back-end setup of the event, to the person at the front-end interfacing with your patrons, the entire system is designed for speed and flexibility using any smart phone or tablet.

The Anywhere Web App

Events and their lists are set up using our web interface, but the power of A-List lies in the phone app and tablet capabilities. Now, in real time, event managers can edit the guest list, parking list, or any other list as needed with a few clicks on their device, for immediate accessibility to all front-of-house employees at all event access points.


No more radioing, texting, or running to the gate and hoping that your list addition or list change gets recorded. Using the app, you can make real time changes to any list within your event, which immediately populates to all devices at all access points.
